Playstation 2 FishEye

Feb 2nd, 2012

We were tasked with creating a ‘visual effect’ for the PS2 using its VCL code. This was quite interesting, I thought I’d found all the ways to crash a PS2, then another one comes along. Get your VCL code wrong, and boom!

I ended up thinking of it as a vertex shader, and created a fish eye type effect. Its not a true fish eye effect because I am actually moving the vertices which means you can peek around corners so to speak.
